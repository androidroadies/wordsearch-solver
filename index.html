wordsearch-solver

wordsearch-solver is a Java library for solving wordsearch puzzles.

Usage:

Sample usage:

String[][] puzzle = ...
BasicGrid grid = new BasicGrid(); // You can create your own implementation of grid if your puzzle is not from a String[][]
WordsearchSolver solver = new WordsearchSolver(grid);
solver.findWord("cheese");
